This project is a fully functional frontend-only URL shortener demo using localStorage as a mock backend.
Features:
- Create short links with optional custom alias (alias uniqueness checks)
- Redirect route that records click events (timestamp, referrer, userAgent)
- Dashboard with list of links, delete, copy and quick analytics
- Analytics page with timestamps, referrer breakdown and device detection
- User auth (register/login/logout) stored in localStorage (demo)

Run:
1. npm install
2. npm run dev
