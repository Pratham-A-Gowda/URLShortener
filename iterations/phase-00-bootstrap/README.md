# Phase 0 – Bootstrap

This phase sets up the minimal folder structure with an empty backend and a Vite + TailwindCSS React frontend scaffold.

## Structure

- backend/ — empty placeholder (no code yet)
- frontend/ — Vite + Tailwind project skeleton

## Frontend

Included:

- Vite config with React plugin
- Tailwind + PostCSS configuration
- Minimal index.html and src/main.jsx using Tailwind utilities

Run locally (optional):

1. `cd frontend`
2. `npm install`
3. `npm run dev`
